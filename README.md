## 2. 技术选型

 - Vue.js / Vue-router / Vuex
 - axios
 - Vant组件库
 - Rem适配
 - 模块化、组件化

## 3. 项目简介

 - 本项目基于Vue全家桶+axios，利用vant组件库，渲染组件；利用vuex数据状态共享，$emit子父传值,props父子传值，解决组件间通信问题，最后实现了一个类似头条的Feed信息流产品软件

## 4. 项目职责

 - 使用EventBus解决了新闻列表组件间对文章不感兴趣和举报的操作
